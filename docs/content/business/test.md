# IslandsQuickEntry 快捷入口

从 Figma 链接 `👻 Islands / QuickEntry` 生成，包含快捷入口、标题、描述等组件。

支持四种变体：
- **数量=1**：单行横向滚动（适合小程序 scroll-view）
- **数量=2**：双列并排
- **数量=3**：三列并排
- **数量=4~9**：网格布局（每行3个）
