# Island Detail 测试页

从 Figma 链接生成的岛屿详情页面，包含以下组件：

- **IslandsHeader** - 头部导航栏（状态栏 + 搜索栏）
- **IslandsPin** - 岛屿固定区（横向滚动）
- **IslandsGrid** - 网格入口（图鉴、角色、潮玩系列）
- **IslandsSlide** - 滑动展示区（Tag 和 SPU）
- **IslandsQuickEntry** - 快捷入口区域
- **IslandsFeed** - Feed 流（双列瀑布流）
- **ButtonFAB** - 悬浮发布按钮
- **TabBar** - 底部导航栏
